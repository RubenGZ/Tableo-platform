## Principios de diseГұo

1. **El engine nunca importa adaptadores directamente** вҖ” usa el factory
2. **Toda tabla tiene RLS** вҖ” sin excepciones
3. **Archivos < 200 lГӯneas** вҖ” si crece, extraer responsabilidad
4. **Sin condicionales por sector fuera de los adaptadores**
5. **Two-phase booking para toda operaciГіn de reserva** вҖ” nunca INSERT directo
